  • STEP 1: Register on codalab

Note: make sure to use the same e-mail address and teamname in the misp2022 and codalab, or you may not be allowed to submit the results in the competition. If you'd like to change e-mail address, please contact with organizers.

  • STEP 2: Apply to join the competition and wait for the approval. Here are Evaluation Phase Leaderboard Entrance (the final Leaderboard )

Tips: just check the box ( you don't need to download the term as you have signed it when you register on misp2022 competition ), and we'll permit access that night.

  • STEP 3: Submit your result and check the scores

  • Make sure your submitted zip file as following directory structure:

    |summission.zip

    ​ ||--R10_S186187188_C11_I1.txt

    ​ ||--*.txt

  • Make sure that all text files named after session are included in the submitted zip package.

  • Make sure each row of the transcript in the following format, which is a key-word separated by a space character:

segment_id 文字文字文字 （only Chinese characters included in the text part without any punctuation）

the segment id is named as follows:

< Speakers ID >_< Room ID >_< Speakers IDs >_< Configuration ID >_>< Index >

  • Here are the submission example and the whole segment_id list for the evaluation set. Notice: you can use your own Speakers ID to replace our global Speakers ID.

  • When the "Status" in the submission table turns to "Finished", you can refresh the page the check the score below. You can also click the button named “View scoring output log” to check if you miss any segments in your submission, for example:

cpCER: 71.11

  • Other information

  • Make sure a. your submitted zip package, includes all sessions in the evaluation set. b. your submission files are in required format, or the submission will fail.

  • For each submitted score, it will be directly on the leader board if it's your best score so far.

  • During the evaluation set phase, every group is allowed to 5 times per day.
