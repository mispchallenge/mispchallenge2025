Submission

Note:

Now Evaluation Set Phase: 2025.01.03 — 2025.01.24 (AOE)
1.Follow the steps mentioned below, make sure to use the same e-mail address on misp2025 to register on codalab, and then send applications to participate.
2.Follow the steps mentioned below, set your teamname on codalab，and make sure it's the same as the one that you use to register in the misp2025.

  • STEP 1: Register on codalab

Note: make sure to use the same e-mail address and teamname in the misp2025 and codalab, or you may not be allowed to submit the results in the competition. If you'd like to change e-mail address, please contact with organizers.

  • STEP 2: Apply to join the competition and wait for the approval. Here are Evaluation Phase Leaderboard Entrance

Tips: just check the box ( you don't need to download the term as you have signed it when you register on MISP2025 competition ), and we'll permit access that night.

  • STEP 3: Submit your result and check the scores

  • Make sure your submitted zip file as following directory structure:

    |submission.zip

    ​ ||--submission.rttm

  • Make sure every row of the transcript use the following format, which is a key-word separated by a space character:

SPEAKER < Session_ID > 1 < start_time > < duration_time > <NA> <NA> < speaker_ID > <NA> <NA>

One example:

SPEAKER M005 1 2.8955 0.9352 <NA> <NA> 10 <NA> <NA>

  • Here is a submission example. Please note that you need to save the results of all sessions into one rttm file..

  • When the "Status" in the submission table turns to "Finished", you can refresh the page the check the score (DER) below. You can also click the button named “View scoring output log” to check the detailed score (FA, MISS, SPKERR).

  • Other information

  • Make sure your submitted rttm file, includes all sessions in the evaluation set and your submission file ia in required format, or the submission will fail.

  • For each submitted score, it will be directly on the leader board if it's your best score so far.

  • During the evaluation set phase, every group is allowed to 5 times per day. Due to the problem of CodaLab, there is a small probability that the result will not be obtained for a long time. Please wait patiently or submit at another time.

  • If you meet any problems during submission, please us by e-mail at mispchallenge@gmail.com.
